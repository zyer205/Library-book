## 3. 未接入的逻辑（原型阶段不做）

以下逻辑在本期静态原型中未实现，留待接后端时处理：

| 逻辑 | 原因 | 当前处理方式 |
|------|------|------------|
| Session 登录态 | 无后端，无法写入 Session | 顶部栏固定显示"学生 A"，通过 JS alert 模拟 |
| 预约冲突检测 | 需后端查询 Reservation 表 | 在预约提交页固定显示冲突提示作为异常状态演示 |
| 时段占用实时计算 | 需后端根据当前时间 + 预约数据计算 | 座位详情页时段网格为静态硬编码 |
| 楼层/区域筛选 | 需 JS 过滤或后端查询 | 筛选下拉框存在但无筛选逻辑 |
| 取消预约 | 需 POST 请求更新数据库 | 使用 `confirm()` + 前端 DOM 删除模拟 |
| 标记完成/取消 | 需 POST 请求更新数据库 | 使用 `alert()` 模拟反馈 |
| 新增/编辑座位 | 需 POST 请求写入数据库 | 弹窗 + `alert()` 模拟 |
| 统计页数据计算 | 需 SQL 聚合查询 | 所有数字为静态假数据 |
| 登录验证 | 需查询 Admin 表 | 表单提交后 `alert()` 模拟 |

## 4. 视觉方向与界面风格

### 本轮采用的视觉方向

**安静学习感（Quiet Study Atmosphere）**

以"图书馆的安静、秩序、专注"为核心情绪，克制使用色彩和装饰，强调信息层级和留白。

### 用户端界面风格

| 维度 | 风格 |
|------|------|
| 布局 | 手机竖屏优先，`max-width: 480px` 居中容器 |
| 导航 | 底部固定 Tab 栏（首页 / 座位列表 / 我的预约），三栏可点击切换 |
| 顶部栏 | 左侧 Logo + 右侧用户信息 + 切换入口 |
| 内容区 | 卡片式布局，圆角 8px，极轻阴影，信息区块分明 |
| 配色 | 主色 #2563EB，辅以功能色（绿/琥珀/红）表达状态 |
| 留白 | 充足上下间距，每个区块之间 12px 分隔 |

### 管理端界面风格

| 维度 | 风格 |
|------|------|
| 布局 | 桌面优先，左侧固定侧边栏 240px + 右侧内容区自适应 |
| 导航 | 侧边栏三栏（预约管理 / 座位管理 / 统计），激活态左侧蓝色竖线 |
| 顶部栏 | 页面标题 + 右侧信息/操作区域 |
| 内容区 | 表格 + 筛选区 + 操作按钮，功能优先 |
| 配色 | 侧边栏深色 #1E293B，内容区浅色背景 #F8FAFC |
| 手机适配 | 不做专门适配，仅确保页面不崩坏 |

## 7. 页面间跳转链路验证

以下核心链路可通过原型中的相对链接点击走通：

### 用户主链路（可操作）

```
index.html → seats.html → seat-detail.html → reservation-create.html → my-reservations.html
    ①          ②              ③                   ④                     ⑤
```

| 步骤 | 操作 | 跳转方式 |
|------|------|---------|
| ① | 首页点击"预约座位" | 链接跳转 |
| ② | 座位列表点击任意座位 | 链接跳转 |
| ③ | 详情页点击"预约此座位" | 链接跳转 |
| ④ | 预约提交页点击"提交预约" | alert 模拟成功 |
| ⑤ | 手动导航到"我的预约" | Tab 栏点击 |

### 管理主链路（可操作）

```
admin/login.html → admin/reservations.html → admin/seats.html → admin/statistics.html
    ①                   ②                       ③                  ④
```

| 步骤 | 操作 | 跳转方式 |
|------|------|---------|
| ① | 登录页输入账号后点"登录" | alert → 手动跳转 |
| ② | 侧边栏点击"座位管理" | 链接跳转 |
| ③ | 侧边栏点击"统计" | 链接跳转 |
| ④ | 侧边栏点击"预约管理"返回 | 链接跳转 |

### 跨端跳转

| 从 | 到 | 方式 |
|----|----|------|
| 管理端登录页 | 用户端 | 底部"← 返回用户端"链接到 `../seats.html` |
| 用户端顶部栏 | 切换用户 | 点击"切换"跳转到 `index.html` |
